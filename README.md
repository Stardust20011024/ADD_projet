# 项目简介
本项目是M2上学期数据分析课程期末小组合作项目

本研究分析了一个包含 30,000 名台湾信用卡客户的数据集，目标是预测信用卡违约风险。我们应用了课程中讲授的多种统计方法，包括：描述性统计分析、主成分分析（PCA/ACP）、逻辑回归、方差分析（ANOVA）以及线性回归（简单回归与多元回归）。

研究结果表明，上一个月的还款状态（PAY_0）是预测违约行为的最重要变量。逻辑回归模型表现出可接受的区分能力，其 AUC 约为 0.73，但在标准阈值 0.5 下，模型的敏感度较低（约为 25%）。

主成分分析结果显示，该数据集主要由两个潜在维度构成：负债水平与偿还能力。由于数据不满足正态性和同方差性等基本假设，线性回归模型在本研究中的适用性相对较弱。
